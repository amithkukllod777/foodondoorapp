# Nutriwow Docs

Yahan Nutriwow store se related saari documentation rehti hai. Folder-wise structure:

| Folder | Kya rakhna hai |
| --- | --- |
| [`company/`](./company) | Company info, policies, brand guidelines, business details |
| [`marketing/`](./marketing) | Campaigns, offers, social media, SEO notes, ad copy |
| [`technology/`](./technology) | Tech stack, architecture, infra, runbooks, integration docs |
| [`operations/`](./operations) | Fulfilment, inventory, shipping, CX operations |
| [`finance/`](./finance) | Financial goals, margins, cost structure |
| [`hr/`](./hr) | Team structure, org, hiring |
| [`legal/`](./legal) | Compliance checklist, statutory, policies |
| [`decisions/`](./decisions) | Decision logs / ADRs — kyun aur kab kya decide hua |
| [`reports/`](./reports) | Sales, analytics, performance aur periodic reports |
| [`agents/`](./agents) | AI agents ke prompts, configs aur usage notes |

> Naya doc add karte waqt usko sahi folder me rakhein. Detail folder ke andar wali README me hai.
